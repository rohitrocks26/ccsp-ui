Modal

Modal Component

Modal Component can be used to give a confirmation message to user regarding the particular action . for example form getting successfully submitted.

Features

  • Uses : Modal can be used to disaply an error or warning ,or it can even be used to take the user input .
  • Values :Modal have the header and body part which are configurable based on the values passed to it .

How to include it

Selector : app-modal

Button used for initiating the modal

Code Snippet

Selector: <app-modal [modeldata]="modModalModule"><app-modal>
