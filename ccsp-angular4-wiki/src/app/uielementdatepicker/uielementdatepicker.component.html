DatePicker

Date Picker Component

This is an injectable component whose properties are purely configurable based on the kind of requirement !!

Features

  • Date Type: User can select the date from the date picker in format of DD/MM/YYYY .
  • Option : User even have an option to change the month , year etc in the date picker

How to Include It

Selector: app-uielementdatepicker

Date Picker Styles

Code Snippet

Selector: <app-uielementdatepicker><app-uielementdatepicker>
