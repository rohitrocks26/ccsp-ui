Angular best Practices

  • Use Angular CLI . For more details please follow this : Angular CLI Library
  • Use Visual Studio Code as it is an awesome IDE for building Angular apps
  • The Angular Language Service — Provides template-aware and Angular-aware completion and error checking withing your application
  • Don’t Subscribe to Your Observables In Components -The better way to do this is to use the async pipe within your templates, and let Angular manage the whole thing for you
  • Don’t Forget about SEO and Analytics
