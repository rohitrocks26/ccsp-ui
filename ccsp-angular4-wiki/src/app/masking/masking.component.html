Masking

Masking Component

This is an injectable directive whose properties are purely configurable based on the kind of requirement !!

Features

  • Data Type : Masking enabled component will be expecting the numeric values from the user.
  • Currency Format : Masking will be done in USD format.
  • Decimal : It does accept the fraction part and mask it accordingly .

How to Include It

Selector: CurrencyFormatter

Masking used inside a input element

Code Snippet

<input type="text" class="form-control" id="donation" [ngModel]="donation" CurrencyFormatter >
